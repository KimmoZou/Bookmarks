给 pdf 添加书签；
- 在界面目录树编辑器编辑当前的书签，
- 导出当前文件的书签
- 导入外部修改好的书签到目录树编辑器
- 保存目录树编辑器的内容到文件副本中

### 运行

安装依赖：
```shell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/ 
```

直接运行 main 文件夹下的 main.py 文件
